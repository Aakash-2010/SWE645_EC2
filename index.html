AI • HEALTHCARE

PPXAI - Privacy Preserving Explainable AI

Overview

PPXAI is a cutting-edge healthcare AI system that addresses one of the most critical challenges in medical AI: balancing the need for accurate, interpretable predictions with strict patient privacy requirements. This project demonstrates how explainable AI techniques can be integrated with privacy-preserving methods to create trustworthy healthcare applications.

Key Features

  • Integration of GRAD-CAM for visual explanations of model predictions in medical imaging
  • Privacy-preserving techniques ensuring sensitive patient data remains protected
  • Explainable AI methods that improve transparency and build trust with healthcare professionals
  • Real-time interpretable predictions suitable for clinical decision-making
  • Compliance with healthcare data privacy standards and regulations

Technologies Used

GRAD-CAM: Gradient-weighted Class Activation Mapping for visualizing which regions of medical images influence model decisions
Explainable AI (XAI): Techniques to make AI decisions interpretable for medical professionals
Privacy Preserving AI: Methods to protect patient data while maintaining model performance
Python: Core development language with deep learning frameworks

Impact

This project contributes to the critical need for trustworthy AI in healthcare by ensuring that AI models not only provide accurate diagnoses but also explain their reasoning in a way that healthcare professionals can understand and verify, all while maintaining the highest standards of patient privacy.

Image Super Resolution using GANs

Overview

This project leverages the power of Generative Adversarial Networks (GANs) to transform low-resolution images into high-resolution outputs with impressive visual quality. Completed during my IBM SkillsBuild internship, this project demonstrates practical applications of deep learning in image enhancement.

Key Features

  • GAN architecture trained to generate realistic high-resolution images from low-res inputs
  • Real-time image enhancement capabilities through Streamlit web interface
  • Perceptual loss functions to maintain visual fidelity and natural appearance
  • Scalable architecture capable of handling various image types and resolutions
  • User-friendly interface for easy image upload and processing

Technical Implementation

GANs: Generator-Discriminator architecture for photorealistic upscaling
TensorFlow: Deep learning framework for model training and inference
Streamlit: Interactive web application framework for deployment
Google Colab: Cloud-based development and training environment
Python: Core programming language with NumPy and OpenCV

SECURITY • WEB APP

Password Manager

Overview

A comprehensive password management solution that combines security best practices with user-friendly design. This full-stack application generates, stores, and manages passwords securely using industry-standard encryption techniques.

Key Features

  • Dynamic password generation with customizable complexity requirements
  • SHA256 encryption algorithm for secure password hashing
  • MySQL database backend for persistent, secure password storage
  • Streamlit-based intuitive user interface
  • Master password protection for accessing stored credentials
  • Password strength analysis and recommendations

Security Features

  • SHA256 cryptographic hashing for password encryption
  • Secure database connections with parameterized queries
  • No plain-text password storage
  • Random password generation with high entropy

Technical Stack

Python: Backend logic and encryption implementation
MySQL: Relational database for secure credential storage
SHA256: Cryptographic hash function for password security
Streamlit: Modern web framework for the user interface
